# ai-influencer-assets

Public CDN-style host for AI influencer persona assets (hero portraits, reference images).

Raw URLs served via `raw.githubusercontent.com/rt45433/ai-influencer-assets/main/<path>` — used as `image_input` for Kie.ai Nano Banana 2 character-anchored generation.

## Personas

- `ellie/hero.png` — Ellie Hart canonical portrait (green eyes, soft brown hair, freckles)
